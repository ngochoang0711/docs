### Architecture Overview

```text
Business Problem / Course Idea
        ↓
BIXSO Orchestrator Agent
        ↓
9 Specialist BA Agents
        ↓
Structured Artifacts
        ↓
Human Review & Decision
```

The **Orchestrator Agent** coordinates the work and distributes tasks.

# The 9 Agents

## 1. Strategy Agent

**Purpose**

Define the **business objective and strategic direction**.

Techniques used

- Impact Mapping
- Business Model Canvas
- SWOT Analysis

Example output

```text
Strategic Goal

Increase BIXSO Academy enrollment by 40% in 12 months.
```

Deliverables

- Strategic goals
- Opportunity analysis
- Value hypothesis

## 2. Market Intelligence Agent

**Purpose**

Understand **industry demand and trends**.

Data sources

- job postings
- industry reports
- community discussions

Example output

```text
Top BA Skills in Demand

• AI-assisted requirements
• Process mining
• Decision modeling (DMN)
• Product discovery
```

Deliverables

- market trend analysis
- competitive benchmarking
- opportunity insights

## 3. Stakeholder Agent

**Purpose**

Identify all **actors influencing the solution**.

Techniques used

- Stakeholder mapping
- Personas
- Organizational modeling

Example output

```text
Stakeholder Map

Primary
• BA professionals
• Corporate clients

Secondary
• training partners
• industry mentors
```

Deliverables

- stakeholder map
- persona definitions
- influence analysis

## 4. Customer Insight Agent

**Purpose**

Understand **user problems and motivations**.

Techniques used

- interview analysis
- journey mapping
- jobs-to-be-done

Example output

```text
Persona: Early-Career Business Analyst

Goal
Learn practical analysis techniques

Pain Point
Courses are too theoretical
```

Deliverables

- personas
- journey maps
- insight reports

## 5. Product Discovery Agent

**Purpose**

Convert insights into **feature ideas and solution concepts**.

Techniques used

- Impact Mapping
- Design Studio
- Concept modeling

Example output

```text
Solution Ideas

• AI-powered BA learning assistant
• Interactive case simulations
• Real-time feedback on analysis exercises
```

Deliverables

- feature concepts
- solution hypotheses
- early prototypes

## 6. Prioritization Agent

**Purpose**

Evaluate which ideas create **the most value**.

Techniques used

- Kano Analysis
- Value vs Effort
- WSJF

Example output

```text
Priority Matrix

High Value
• Interactive case simulations
• AI feedback system

Low Priority
• Dark mode
• Animated tutorials
```

Deliverables

- prioritization matrix
- Kano classification
- feature ranking

## 7. Solution Design Agent

**Purpose**

Transform ideas into **structured requirements**.

Techniques used

- user stories
- process modeling
- data modeling

Example output

```text
User Story

As a BA learner
I want an AI assistant to review my Impact Map
so that I can improve my analysis skills.
```

Deliverables

- user stories
- acceptance criteria
- process diagrams

## 8. Risk & Compliance Agent

**Purpose**

Identify **constraints and risks**.

Focus areas

- regulatory constraints
- technical risk
- project feasibility

Example output

```text
Risk

AI feedback may produce incorrect advice.

Mitigation
Human instructor review.
```

Deliverables

- risk register
- compliance checklist
- mitigation strategies

## 9. Delivery Agent

**Purpose**

Convert the design into a **delivery-ready roadmap**.

Techniques used

- MVP definition
- backlog creation
- release planning

Example output

```text
MVP Scope

• AI case study assistant
• Impact Mapping exercise
• Kano analysis module
```

Deliverables

- MVP definition
- product roadmap
- sprint backlog

# How the Agents Collaborate

The agents follow a **structured pipeline**.

```text
Strategy Agent
      ↓
Market Intelligence Agent
      ↓
Stakeholder Agent
      ↓
Customer Insight Agent
      ↓
Product Discovery Agent
      ↓
Prioritization Agent
      ↓
Solution Design Agent
      ↓
Risk & Compliance Agent
      ↓
Delivery Agent
```

Each step **enriches the analysis**.

Human workflow

```text

      ↓

      ↓
```
