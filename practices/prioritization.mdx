# 1. How the Agentic System Orchestrates the Analysis

The system has an **Orchestrator Agent** that manages the workflow.

The orchestrator acts like a **digital program director**.

`Goal: Design a new BA training program for professionals`

`Target: Junior → Senior Business Analysts`

`Focus: AI-assisted analysis and digital transformation`

### Orchestration Pipeline

```text
Course Idea
     ↓
Orchestrator Agent
     ↓
Specialist Agents
     ↓
Structured Outputs
     ↓
Human Review
     ↓
Final Course Program
```

The orchestrator breaks the project into tasks such as:

| Task                     | Assigned Agent             |
| :----------------------- | :------------------------- |
| Analyze industry demand  | Market Research Agent      |
| Identify skills required | Curriculum Agent           |
| Design learning modules  | Instructional Design Agent |
| Create exercises         | Case Study Agent           |
| Build assessments        | Evaluation Agent           |

# 2. How Multiple AI Agents Collaborate

The agent system behaves like a **virtual education team**.

Each AI agent has a **specific responsibility**.

## 1️⃣ Market Research Agent

Purpose: Identify what skills Business Analysts need in the AI era.

Input:

```text
BA job postings
industry reports
BABOK techniques
AI trends
```

Output

```text
Top Skills Required
• Decision modeling
• Process mining
• AI-assisted requirements analysis
• Data-driven root cause analysis
```

## 2️⃣ Curriculum Design Agent

Purpose: Transform skill requirements into **learning modules**.

Output example

```text
Course Structure

Module 1 – Strategy & Impact Mapping
Module 2 – Customer Discovery
Module 3 – Product Prioritization
Module 4 – AI-Assisted Requirements
Module 5 – MVP & Experimentation
```

## 3️⃣ Case Study Agent

Purpose: Generate real-world exercises.

Example output

```text
Case Study
Digital Transformation for Retail Bank

Students must:
• Build an Impact Map
• Perform Kano analysis
• Define MVP scope
```

## 4️⃣ Content Generation Agent

Purpose: Create course materials.

Output

```text
Slides
Reading material
Templates
Framework explanations
```

## 5️⃣ Assessment Agent

Purpose: Evaluate student work automatically.

Example out

```text
Student Impact Map
Score: 85/100

Strength:
Clear goal definition

Improvement:
Actors missing compliance stakeholder
```

# 4. How Humans Stay in the Decision Loop

Even with an agentic system, **humans remain the decision makers**.

This is called **Human-in-the-Loop governance**.

The workflow looks like this.

```text
AI Analysis
     ↓
Human Review
     ↓
Human Decision
     ↓
AI Refinement
```

## Human Roles in BIXSO Academy

| Role                  | Responsibility            |
| :-------------------- | :------------------------ |
| Academy Director      | Approve course strategy   |
| Lead Instructor       | Validate curriculum       |
| Subject Matter Expert | Review case studies       |
| Students              | Provide feedback          |
| Industry Partners     | Validate skills relevance |

### Example Decision Checkpoint

AI proposes this course:

```text
Module 4:
AI-Generated User Stories
```

Instructor review:

```text
Concern:
Students must learn manual story writing first
```

Decision:

```text
Module 4 split into:

4A Manual Story Writing
4B AI-Assisted Story Generation
```
